# OCR Ready Engine Patch

แพตช์นี้แทน engine OCR ที่เคย patch ต่อท้ายหลายรอบด้วยชุดไฟล์สะอาดชุดเดียว
เพื่อให้ระบบอ่านตามกรอบที่สอนเป็นหลัก และจูน OCR ตามชนิดข้อมูล

## วิธีลง

```powershell
cd C:\Users\Admin\Desktop\TMC_OCR
python apply_ocr_ready_engine_patch.py
python check_ocr_ready_engine.py
python main.py
```

## หลังลงแล้วให้ทดสอบ
1. เปิดโปรแกรม
2. เข้าแท็บประมวลผล PO
3. เลือกลูกค้า
4. กดสอนตำแหน่ง แล้วลากกรอบให้ครบ: ลำดับ, รหัสสินค้า, ชื่อสินค้า, จำนวน, ราคา, จำนวนเงิน
5. เลือก Filter ที่เหมาะ เช่น ลบเส้นสีรบกวน หรือ เอกสารจาง/สีซีด
6. บันทึกโปรไฟล์
7. กลับหน้า Process PO แล้วล้าง OCR Cache ทั้งหมด
8. OCR ใหม่

## จุดสำคัญ
- ระบบจะใช้ template/กรอบที่สอนเป็นหลักก่อนเสมอ
- ช่องตัวเลขใช้ whitelist เฉพาะเลข จุด ลูกน้ำ
- ช่องรหัสสินค้าใช้ whitelist อังกฤษ/ตัวเลข/ขีด/จุด
- ลบเส้นสีรบกวนก่อน OCR
- ซ่อมราคา/ยอดเงินด้วยสูตร amount / qty
- ถ้าอ่านยอดท้ายบิลได้ จะยึดยอดท้ายบิลเป็นหลัก
